Manage access to your Temporal Cloud account and namespaces through various authentication and authorization mechanisms.

## Namespace Authentication

- [**Certificates**](certificates) - Configure mTLS certificates to authenticate to a namespace
- [**API Keys**](api-keys) - Create and manage API keys to authenticate to a namespace

## Identity Management

- [**Users**](users) - Manage individual user accounts and permissions
- [**Service Accounts**](service-accounts) - Configure service accounts for automated access
- [**User Groups**](user-groups) - Organize users into groups for simplified access management
- [**SAML**](saml) - Configure SAML-based single sign-on integration
- [**SCIM**](scim) - Use your IDP to manage Temporal Cloud users and access via SCIM integration
