## Quick start

1. Add the [Temporal Go SDK](https://github.com/temporalio/sdk-go) to your project:

```
go get go.temporal.io/sdk@latest
```

2. [Install and run the Temporal Server](/docs/server/quick-install) using `docker compose`.
