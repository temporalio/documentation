A `tctl admin` command allows the user to run admin operations.

Alias : adm

#### tctl admin commands

- tctl admin workflow
- tctl admin shard
- tctl history_host
- tctl taskqueue
- tctl membership
- tctl cluster
- tctl dlq
- tctl db
- tctl decode

#### Options

show help: `tctl admin [--help | -h]`
