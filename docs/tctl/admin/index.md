A `tctl admin` command allows the user to run admin operations.

Alias : `adm`

#### tctl admin commands

- [`tctl admin workflow`](/docs/tctl/admin/workflow/)
- [`tctl admin shard`](/docs/tctl/admin/shard/)
- `tctl history_host`
- `tctl taskqueue`
- `tctl membership`
- `tctl cluster`
- `tctl dlq`
- [`tctl db`](/docs/tctl/admin/db/index)
- `tctl decode`

#### Options

Help: `tctl admin [--help | -h]`
