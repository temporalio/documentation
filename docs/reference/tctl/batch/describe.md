The `tctl batch describe` command describes the progress of a batch job.

The following modifier controls the behavior of the command.

### `--job_id`

_Required modifier_

How to specify the job ID of a batch job.

Alias: `--jid`

**Example**

```
tctl batch describe --job_id <id>
```
