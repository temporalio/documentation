To send a Signal to a Workflow and start the Workflow if it isn't already running, use `signalWithStart()`.

```typescript
const client = new WorkflowClient();
await client.signalWithStart(YourWorkflow, {
  workflowId,
  args: [arg1, arg2],
  signal: YourSignal,
  signalArgs: [arg3, arg4],
});
```
