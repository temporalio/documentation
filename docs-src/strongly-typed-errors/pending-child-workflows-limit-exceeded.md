title: Pending Child Workflows Limit Exceeded
description: Explanation for Pending Child Workflows Limit Exceeded error message, and how to fix it.
sidebar_label: Pending Child Workflows Limit Exceeded
tags:
  - errors
  - strongly-typed
---

This error indicates that the Workflow Task was failed to prevent the creation of more Child Workflows.

Wait for the system to finish any currently running Child Workflows before retrying this Task.
