The `tctl admin cluster upsert_remote_cluster` command adds or updates remote Cluster information in the current Cluster.

Alias: `urc`

#### --frontend_address

Alias: `--fad value`

The remote Cluster frontend address.

#### --enable_connection

Alias: `--ec`

Enables remote Cluster connection.
