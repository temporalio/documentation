The `tctl admin workflow describe` command describes internal information of the current [Workflow Execution](/workflows/#workflow-execution).

#### --workflow_id value

Alias: `-w`

The Id of the current Workflow.

#### --run_id value

Alias: `-r`

The Id of the current run.
